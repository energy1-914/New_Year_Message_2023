### Vanilla Javascript로 신년메세지 주고받는 페이지를 제작했습니다.
### 총 3 페이지로 포스트 작성 페이지, 포스트 목록 페이지, 포스트 내용 페이지로 이루어져 있습니다.
### 백엔드에서 전해준 url을 통해 fetch api를 사용해 CRUD를 구현했습니다.
### 또한, 포스트 작성 시 랜덤 이미지는 unsplash에서 제공하는 api를 사용했습니다.
